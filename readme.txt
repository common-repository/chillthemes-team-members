Enables a post type to display team members for use in any of our Chill Themes.

== Description ==

This plugin provides you with a new post type aptly named **Team Members** to use with any theme you want, obviously it was created for use in our themes.

== Installation ==

= Using The WordPress Plugin Upload =

1. Log in and navigate to Plugins &rarr; Add New.
2. Type **ChillThemes Team Members** into the Search input and click the **Search Plugin** button.
3. Locate the **ChillThemes Team Members** in the list of search results and click the **Install Now** button.
4. Click the **Activate Plugin** link at the bottom of the install screen.

= Using FTP =

1. Download the **ChillThemes Team Members** plugin from WordPress.org.
2. Unzip the package and move it to your plugins directory.
3. Log into WordPress and navigate to the **Plugins** screen.
